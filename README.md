# To run the project

## Open VS Code
## Go to extensions 
## Install "Live Sever" : "By Ritwick Dey" extension
## After installion is done , proceed to where your project is at
## Right-click the .html file and open it with "Open with Live Server" 
## Or Open the .html file and click the browser icon on the top right corner.

## Install necessary dependencies
npm install

## To start project
### create the .env file :
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=wil_contract_db
DB_PORT=3306
PORT=3000

### run
npm install dev

## run on frontend on port:5500
http://127.0.0.1:5500/client/index.html
